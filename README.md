# SISA Machine Unlearning

Implementation and experiments for SISA (Sharded, Isolated, Sliced, and Aggregated) machine unlearning.

## Contents

- `SISA_Machine_Unlearning.ipynb`: end-to-end notebook for experiments and analysis.
- `src/`: supporting source code for API, SISA engine, and web components.
- `run.py`: project entry-point script.
- `requirements.txt`: Python dependencies.

## Notes

Large datasets, model checkpoints, and generated plots/reports are intentionally excluded from version control.
